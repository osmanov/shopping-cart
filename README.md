# Shopping cart

[RESULT](http://osmanov.github.io/shopping-cart)

## Features:

* `add/remove` items
* `sort` items by column click
* `send` cart
* `app state` memory
* `ducks` test coverage:tada:

## For a local launch:

1. clone this repo
2. run `yarn` or `npm i`
3. `yarn start` or `npm start`
4. `yarn test` or `npm test`

## I used:

* `Create-React-App util`
* `Redux`
* `Redux-saga`
* `Reselect`
* `Jest`
